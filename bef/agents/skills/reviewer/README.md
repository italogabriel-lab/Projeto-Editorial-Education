## Objetivo

---

## Comandos

### `/review <arquivo>`

**Descrição:** Revisa arquivo formatado.

**Sintaxe:**
```bash
/review formatted_class.md
```

---

## Checklist de Auditoria

### 0. Título H1
- [ ] Idêntico ao Currículo Macro

### 1. Estrutura
- [ ] Voz ativa
- [ ] Imperativo direto
- [ ] Tempo presente

### 2. Formatação
- [ ] Rise Blocks corretos
- [ ] Sem H3 (usar `[+HEADING]`)
- [ ] Aspas retas

### 3. Conteúdo
- [ ] 5 Hábitos completos
- [ ] Definições idênticas
- [ ] Cosmovisão reformada

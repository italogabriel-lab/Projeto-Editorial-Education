# Story 0.1: Estruturar o TriviumOS como framework instalável

## Status

Done

## Story

**As a** líder da operação editorial,  
**I want** transformar o fluxo editorial atual em um framework instalável com a arquitetura do AIOX,  
**so that** toda a equipe possa replicar a mesma metodologia em outras máquinas e bases disciplinares.

## Acceptance Criteria

1. O repositório documenta explicitamente os padrões hierárquicos do AIOX e como eles foram adaptados para o TriviumOS.
2. Existe um pacote Node instalável via `npx create-triviumos` que gera a estrutura-base do framework.
3. O framework instalado contém `.triviumos-core`, registries, agentes, workflows, checklists e documentação.
4. Todos os agentes, templates, workflows e scripts editoriais existentes estão inventariados e organizados em uma taxonomia única.
5. As lacunas funcionais encontradas no inventário atual foram corrigidas ou documentadas com clareza.
6. O projeto passa por `npm run lint`, `npm run typecheck` e `npm test`.

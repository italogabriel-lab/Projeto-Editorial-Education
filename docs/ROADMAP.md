**Objetivo:** Integrar os dados brutos e garantir o provisionamento visual.

**Objetivo:** Transformar o Dashboard em Ferramenta Operacional e Preditiva.
- [ ] Construir drill-down (Cliquem no Gráfico e vejam os tickets detalhados).

- [ ] Visualização histórica (Evolução do projeto mês a mês).

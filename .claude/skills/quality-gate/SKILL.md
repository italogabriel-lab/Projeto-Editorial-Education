# Skill, gate de qualidade

Use esta skill antes de concluir uma aula, revisão, prova ou publicação.

## Acionamento

- "Revise esta aula"
- "Faça QA"
- "Posso publicar?"

## Procedimento

1. Ler `trivium-method-editorial/agents/skills/reviewer/SKILL.md`.
2. Validar título, estilo, doutrina, Rise Blocks e consistência da definição.
3. Verificar aderência ao golden template.
4. Identificar erros bloqueantes antes da publicação.

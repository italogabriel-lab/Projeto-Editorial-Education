# Skill, sincronização de títulos

Use esta skill quando houver divergência entre título de aula, arquivos estruturais e Currículo Macro.

## Acionamento

- H1 diferente do Currículo Macro
- Ajustes em matriz, visão ou descrições
- Pedido de revisão estrutural de nomenclatura

## Procedimento

1. Localizar o `1 - Curriculo Macro` do ano correto.
2. Comparar títulos nos arquivos dependentes.
3. Executar `trivium-method-editorial/scripts/sync_titles.py`.
4. Revisar o resultado.
5. Confirmar que H1, matriz, visão e descrições ficaram sincronizados.

## Resultado esperado

Uma base curricular coerente, sem títulos divergentes.

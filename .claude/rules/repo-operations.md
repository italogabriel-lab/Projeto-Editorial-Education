# Operação do repositório

## Convenções

- Preferir commits convencionais.
- Preservar mudanças locais do usuário.
- Não sobrescrever conteúdo editorial sem validar contexto.
- Consultar a knowledge base antes de alterar comportamento do framework.

## Locais importantes

- Framework, `trivium-method-editorial/`
- Conteúdo publicado, `Projeto - Bibline Academy ( Produção de Aulas)/`
- Guias operacionais, `docs/process/`
- Arquitetura do framework, `docs/framework/`

## Quando usar scripts

- `trivium-method-editorial/scripts/sync_titles.py`, divergência de títulos
- scripts de formatação e manutenção dentro de `trivium-method-editorial/scripts/`, ajustes estruturais do framework

## Publicação

Antes de publicar, confirmar revisão, consistência curricular e caminho correto do arquivo final.

# Contexto do projeto

## Identidade

Plataforma de produção de conteúdo didático para Educação Clássica Cristã Reformada.
O núcleo do framework está em `trivium-method-editorial/`.

## Fonte de verdade

- Skills, `trivium-method-editorial/agents/skills/`
- Base de conhecimento, `trivium-method-editorial/knowledge-base/`
- Templates, `trivium-method-editorial/templates/`
- Workflows, `trivium-method-editorial/workflows/`
- Scripts utilitários, `trivium-method-editorial/scripts/`

## Estrutura principal do repositório

- `.claude/`, instruções do Claude e comandos
- `trivium-method-editorial/`, framework editorial
- `Projeto - Bibline Academy ( Produção de Aulas)/`, conteúdo educacional
- `docs/`, documentação de arquitetura, operação e roadmap
- `scripts/`, automações de suporte
- `public/`, `src/`, `scaffold/`, camada de produto e scaffolder

## Idioma

Todas as interações e saídas devem ser em português.

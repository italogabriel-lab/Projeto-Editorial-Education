## 🔍 FORMATO DE CADA HÁBITO

### 1. DEFINIR

• Apresente a definição do Webster’s 1828 (em inglês e português).  
• Destaque palavras-chave.  
• Relacione a definição a um versículo.  
• Consulte o PDF e site do Webster 1828.  
• Use blocos Rise 360: Paragraph, Video, Heading.  
• Proponha atividade com blocos: Accordion, Tabs.

### 2. PERCEBER

• Crie comparação com imagens, símbolos cristãos ou natureza.  
• Estimule observação visual e contemplação.  
• Use o PDF Encanto Família de Trigo e imagens do site ARC Acervum.  
• Aplique blocos Rise 360: Image Labeled, Carousel, Process, Heading.

### 3. RECORDAR

• Reforce vocabulário com flashcards, poemas ou músicas cristãs.  
• Use referências do Encanto Família de Trigo e materiais clássicos.  
• Aplique blocos Rise 360: Flashcards, Statement D, Image Text On.

### 4. PRATICAR

• Proponha tarefa concreta manual, artística ou lógica.  
• Fundamente cada prática com base bíblica.  
• Inspire-se em Charlotte Mason.  
• Busque referências no site ARC Acervum.  
• Use blocos Rise 360: Fill-in, Sorting, Matching, Paragraph.

### 5. NARRAR

• Escolha narrativa cristã, livro vivo ou fábula clássica.  
• Reorganize o conteúdo com três perguntas imperativas: o que aconteceu, qual a moral teológica, como se conecta ao termo.  
• Finalize com versículo de memorização.  
• Use recursos: Livros Coloridos, Livro das Virtudes, obras de Charlotte Mason.  
• Aplique blocos Rise 360: Heading, Statement A, Paragraph, Image Aside.

## Objetivo

Criar provas bimestrais completas com base no conteúdo das 8-10 semanas de cada bimestre.

---

## Comandos

### `/exam create <bimestre> <ano> <disciplina>`

**Descrição:** Cria prova bimestral.

**Sintaxe:**
```bash
/exam create <bimestre> <ano> <disciplina>
```

**Exemplo:**
```bash
/exam create 2 3 "Matemática"
```

---

## Estrutura da Prova

### Questões por Tipo
- **Fill-in:** 5-10 questões
- **Multiple Choice:** 5-10 questões
- **True/False:** 5 questões
- **Matching:** 3-5 questões

### Cobertura
- Todas as semanas do bimestre
- Todos os 5 hábitos
- Termos principais

**Descrição:** Aplica capitalização correta.

```bash

```

---

## Regras de Capitalização

### ✅ Maiúscula
- Primeira palavra da frase
- Nomes próprios
- Instituições consolidadas

### ❌ Minúscula
- Termos descritivos
- Direções geográficas
- Períodos históricos (genéricos)

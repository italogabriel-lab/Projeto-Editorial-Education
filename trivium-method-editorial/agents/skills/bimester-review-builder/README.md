## Objetivo

Criar revisões bimestrais que consolidam todo o conteúdo das 8-10 semanas de cada bimestre.

---

## Comandos

### `/review create <bimestre> <ano> <disciplina>`

**Descrição:** Cria revisão bimestral.

**Sintaxe:**
```bash
/review create <bimestre> <ano> <disciplina>
```

**Exemplo:**
```bash
/review create 2 3 "Ciências"
```

---

## Estrutura da Revisão

### Seções
1. **Definições:** 9 termos principais
2. **Conceitos:** Ideias-chave do bimestre
3. **Conexões:** Relações entre semanas
4. **Aplicações:** Uso prático do conteúdo

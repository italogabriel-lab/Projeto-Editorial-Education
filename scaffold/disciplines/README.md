# Disciplines

Use este diretório para plugar a base de conhecimento específica de cada disciplina da equipe.

Estrutura recomendada:

```text
disciplines/
  belas-artes/
    knowledge-base/
    curriculum/
    reports/
  matematica/
    knowledge-base/
    curriculum/
    reports/
```

O framework TriviumOS separa:

- biblioteca editorial compartilhada em `triviumos/`
- governanca e runtime em `.triviumos-core/`
- conhecimento específico da disciplina em `disciplines/`
